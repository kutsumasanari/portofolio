製品の在庫管理API
====

製品の在庫管理の FastAPI を作成した。製品名（product_name）、製品価格（price）といった情報を登録（create）・参照（read）・更新（update）・削除（delete）できた 。操作に不備があった場合、エラーコードを表示した。
さらに、このFastAPIをAWSにデプロイした。
